## 🚀 주요 구현 내용

### 1. 동적 채굴 레이캐스트 (Dynamic Line Trace)

- **방향 인식 시스템**: 캐릭터가 이동한 마지막 방향(`GetLastMovementInputVector()`)의 X축 값을 판별하여, 제자리에 있을 때는 아래(`Z: -1`)로, 좌우 이동 중일 때는 해당 방향(`X: ±1`)으로 레이캐스트를 발사하도록 구현했습니다.
- **디버그 라인**: `#if WITH_EDITOR` 전처리기를 사용하여 에디터 모드에서만 빨간색 레이캐스트 선이 보이도록 처리하여 디버깅을 용이하게 했습니다.

### 4. PaperZD 애니메이션 블루프린트(ABP) 동기화

- 캐릭터 캐싱 : 매번 플레이어 오너를 가져오면 리소스를 많이 사용하므로 이벤트 그래프에서 `Event Onlnit` → `Cast To DrillerCharchter`(`GetPaperCharacter`로 밸류 가져오기) → `Set` 으로 가져와서 사용
    - 유니티로 비유하면 시작 시 겟 컴포넌트로 클래스를 담아서 사용한다고 보면됨
- **이벤트 기반 점프 (Jump to Node)**: 매 프레임 C++에서 `bIsDigging` 상태를 가져와 `Branch`로 검사한 뒤, `Do Once` 노드를 통해 딱 한 번만 채굴 애니메이션을 트리거하도록 최적화했습니다.
- **자동 리셋 (Animation Notify)**: 채굴 플립북의 마지막 프레임에 `EndDig` 노티파이를 추가하고, ABP 이벤트 그래프에서 이 노티파이가 터지면 C++의 `StopDigging()`을 호출하여 캐릭터를 다시 Idle 상태로 자연스럽게 돌려보냅니다.
- **방향별 모션 분기 (Select 노드)**: `bisDiggingSideABP` 변수와 `Select` 노드를 활용해, 아래로 팔 때는 `DigDown`, 옆으로 팔 때는 `DigSide` 점프 노드로 영리하게 분기시켰습니다.

### 2. 블록 파괴 최적화 (Object Pooling)

- 블록(`ABlock`)을 파괴할 때마다 `Destroy()` 함수를 호출하면 발생하는 **가비지 컬렉션(GC) 부하 및 프레임 드랍을 방지**하기 위해, 블록을 비활성화(`Hidden`, `No Collision`)하여 풀(Pool)로 되돌려 보내는 방식을 적용했습니다.

### 4. 뷰포트 & 카메라 개선

- 레벨 블루프린트(Level BP)에 하드코딩 되어있던 기존 카메라 로직을 삭제했습니다.
- 대신 `ADrillerCharacter`에 **Spring Arm**과 **Camera 컴포넌트**를 부착하여, 캐릭터가 깊은 땅속으로 파고 내려갈 때 부드럽게 화면이 따라가도록(Camera Lag 활용 가능) 구조를 개선했습니다.

## 💻 핵심 코드 스니펫

### 1. C++: 방향 인식 채굴 로직 (`ADrillerCharacter::Dig`)

```cpp
void ADrillerCharacter::Dig()
{
    StartDigging(); // 애니메이션 트리거 (bIsDigging = true)
    
    FVector rayStart = GetActorLocation();
    FVector digDirection = FVector(0.f, 0.f, -1.f); // 기본: 아래
    FVector lastInput = GetLastMovementInputVector();
    
    // 이동 키 입력에 따른 방향 덮어쓰기 및 사이드 채굴 상태 저장
    if (!FMath::IsNearlyZero(lastInput.X))
    {
       float dirX = FMath::Sign(lastInput.X);
       digDirection = FVector(dirX, 0.f, 0.f);
       bIsDiggingSide = true; 
    }
    else
    {
       bIsDiggingSide = false;
    }

    FVector rayEnd = rayStart + (digDirection * 42.f); 
    FHitResult hit;
    FCollisionQueryParams params;
    params.AddIgnoredActor(this);
    
    bool bHit = GetWorld()->LineTraceSingleByChannel(hit, rayStart, rayEnd, ECC_Visibility, params);
    
    // 블록 감지 시 상호작용
    if (bHit)
    {
       ABlock* HitBlock = Cast<ABlock>(hit.GetActor());
       if (HitBlock != nullptr)
       {
          HitBlock->OnInteracted(this); // 풀링 비활성화 로직 실행
       }
    }
}
```
